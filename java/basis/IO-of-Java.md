Java中的IO种类看似繁杂，实则有规律可循。

按照功能可以分为：

1. 缓冲区
2. memory
3. 文件
4. Data Conversation
5. Object
6. 管道pipe
7. print，用于打印
8. peeking ahead，预读
9. 流合并
10. filter
11. 数据流转换，从字符流转到字节流
12. 按行读取，建议使用字符流，字节流已过时
